Key Innovations

Lessons learned from real-world desktop automation

ğŸšï¸ UIA Slider Control

Direct RangeValuePattern.SetValue() via Windows UI Automation. 100% reliable vs unreliable mouse drag on slider controls. Discovered while debugging Demo 1 (brightness).

ğŸ” Content Verification

Post-click window title checking prevents wrong content selection. Essential for YouTube video search â€” the agent verifies the video matches the search query before proceeding.

ğŸ”„ Recovery-Aware Actions

Auto-recovery (Escape key) can sabotage in-progress operations like folder rename dialogs. Solution: per-app recovery strategy that knows when NOT to intervene.

âœ… Filesystem Verification

Before accepting "done", verify the actual outcome on disk. For Demo 4: check that ~/Downloads/TestFromAgenticOS exists. Prevents false success claims.

ğŸ‘¤ Human Signal > Automated Metrics

First supervised run revealed: demos that pass automated checks (return success=true) may still fail human quality expectations. Human RL signal catches nuances automated metrics miss.

âš¡ Speed â‰  Mouse Speed

Optimization must target overhead (LLM latency, UIA detection, validation) not physical interaction speed. Human reviewer explicitly requested: "slow down input to human speed."
